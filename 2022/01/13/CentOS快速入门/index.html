# CentOS 目录结构

bin 存放二进制可执行文件（ls,cat,mkdir 等）

etc 存放系统配置文件

home 用户目录

usr 默认安装软件目录

var 存放运行时需要改变数据的文件

# 操作文件

命令 说明 语法 参数 参数说明
ls 显示文件和目录列表
-l 列出文件的详细信息
-a 列出所有文件，包含隐藏文件
mkdir 创建目录 mkdir [-p] dirname
-p 创建多层目录
cd 切换目录 cd [dirname]
touch 生成一个文件
echo 生成一个带内容的文件 echo aaa>a.txt,echo bbb>>a.txt
cat 显示文本内容 cat filename
cp 复制文件或目录
rm 删除文件 rm name
-f 强制删除文件或目录
-r 同时删除该目录下所有文件
mv 移动文件或目录
find 在文件系统中查找指定的文件 -name 文件名
grep 在指定的文本文件中查找指定的字符串
pwd 显示当前工作目录
ln 建立软链接
more 分页显示文本文件内容
head 显示文件开头内容
tail 显示文件结尾内容

# 压缩解压缩

命令 语法 参数 参数说明
tar tar [-cxzjvf] filename/dirname
-c 打包参数指令
-x 解压参数指令
-z 是否需要用 gzip 压缩
-j 是否需要用 bzip2 压缩
-v 压缩过程中显示文件
-f 解压缩名称
-tf 查看压缩文件里面的文件

# Linux 系统管理

命令 说明
stat 显示指定文件的相关信息
who 显示在线登录用户
hostname 显示主机名称
uname 显示系统信息
top 显示系统中各个进程的资源占用状况
ps 显示当前的进程状态
du 显示指定的文件（目录）已使用的磁盘空间的总量
df 显示文件系统磁盘空间的使用情况
free 显示当前内存和交换空间的使用情况
ip address 显示网络接口信息
ping 测试网络的连通性
clear 清屏
kill 杀死一个进程

# 开关机命令

重启

reboot

shutdown -r now

关机

shutdown -h now
